Projekt front-endowy odwzorowujący wyszukiwarkę Google: Google Search, Google Image Search oraz Google Advanced Search. 


---

## 🔍 Funkcjonalności

- Strona główna wyszukiwarki Google (Search)
- Wyszukiwanie obrazów (Google Image Search)
- Zaawansowane wyszukiwanie (Google Advanced Search)
- Przycisk „I’m Feeling Lucky” (przekierowanie do pierwszego wyniku)
- Wyszukiwanie oparte o parametry GET (`q=...`)
- Układ i styl inspirowany Google

---

## 🧠 Czego się nauczyłem

- działania formularzy HTML i metody GET
- pracy z parametrami URL (query string)
- budowy wielostronicowych aplikacji statycznych
- podstaw CSS i tworzenia layoutu
- sposobu działania wyszukiwarek internetowych

---

## 🛠️ Technologie

- HTML
- CSS
- podstawy HTTP / GET requests

---

## 📌 Struktura projektu

- `index.html` – strona główna wyszukiwarki
- `image.html` – wyszukiwanie obrazów
- `advanced.html` – wyszukiwanie zaawansowane
- `styles.css` – style strony

---

## 🚀 Jak działa projekt

Formularze wysyłają zapytania bezpośrednio do Google poprzez URL:

Przykład:

https://www.google.com/search?q=car
